Elumia

Designing a game-native staking experience for a Web3 platform where players need to understand rewards, chest upgrades, and market activity without feeling like they are reading a DeFi dashboard.

2023 · UX / UI Design · Web3 · Game Economy

The Situation

The product needed to feel like a game, not a spreadsheet.

Elumia is a Web3 platform built around play-to-earn mechanics, staking, and crypto-game progression. The challenge was to make those systems understandable for players who care about rewards and progression, but do not want to read through a finance-first interface.

Users had to navigate multiple exchanges, staking events, chest tiers, and price references. Without a stronger visual system, the experience risked feeling too technical for a game audience.

The Approach

World-building around the staking flow.

The UI uses darker surfaces, gold accents, and character-led art direction to make the staking journey feel like part of the Elumia universe. The design keeps the core actions visible while framing them in a way that feels closer to a game hub than a trading screen.

Marketplace navigation, staking events, price feeds, and chest upgrades are all organized so players can move from discovery to action without losing the theme. The interface stays functional, but the presentation remains immersive.

The Design

Home page hero and navigation establish the game world immediately, while still making the staking experience easy to enter.

The staking hub keeps rewards, pools, and price references in one view so players can understand the next move quickly.

Staking event screens introduce event-based progression, making each reward path feel like part of the wider game loop.

A second event state keeps the same visual language while showing a different step in the staking and reward flow.

Chest upgrades use a premium game UI treatment, turning a functional mechanic into a clear part of the progression story.

The Results

A staking flow that feels native to the game.

The redesign helps Elumia explain a complex set of staking actions without breaking the fantasy layer that players expect. The product stays usable, but the experience feels far more aligned with the audience.

Game-first presentation

Dark surfaces, gold accents, and illustrated elements make the platform feel like a game ecosystem instead of a DeFi panel.

Clear staking hierarchy

Rewards, event cards, and pricing are organized so players can move through the staking journey without guesswork.

More approachable Web3 UX

Players can understand the action without needing to decode the underlying crypto mechanics first.

Reusable system

The visual language can scale across future staking events, chest upgrades, and marketplace states.

Conclusion

"When the interface feels like the world, the mechanics are easier to trust."

Elumia needed more than a layout pass. It needed a presentation layer that could make Web3 mechanics feel natural inside a game-led product story.
