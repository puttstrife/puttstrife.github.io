Wells Fargo
& Coral Active

Designing an omnichannel agent workspace for a regulated banking environment — where a bad workflow isn't friction, it's a compliance liability.

2024 · UX Lead · Enterprise Banking

The Situation

This one is about risk.

Wells Fargo's call center agents — tellers, back-of-house staff, service agents — were all running through the same platform. These weren't just knowledge workers. In a regulated banking environment, every workflow carries compliance weight.

The problem wasn't interface friction. It was operational risk. A bad workflow in banking doesn't just cost time — it's a liability. Mis-routed interactions, inconsistent authentication steps, and context loss between channels can each trigger audit findings.

Direct user research wasn't possible inside a closed enterprise system. So we triangulated from support tickets, stakeholder sessions, training materials, and operational workflows — building design confidence from proxy insight rather than traditional discovery.

A 5-step structured process: understand the operational system → identify critical workflows → gather proxy insight → turn signals into decisions → align with organisational goals.

The Results

One workspace. Every channel.

The Coral Active platform unified voice, chat, and email into a single browser-based workspace. Agents log in once — their role and location automatically load the correct permissions, embedded within existing legacy infrastructure. No new systems. No retraining.

Unified Availability

Agent sets status once — routing handles all channels automatically. No tool switching, no missed interactions.

Search + Softphone Together

Incoming call opens a search tab alongside the softphone. Customer identified without interrupting the call — context available before the first word.

AI Interaction Summary

Every interaction — voice, chat, email — ends with an AI-generated summary. Agents review and edit before submitting, keeping human judgment in the loop.

AI Assist

Real-time response suggestions and next-action prompts reduce typing and cognitive load during live interactions — agents stay focused on the customer.

Inline 2FA for Chat

Authentication handled inside the conversation — no navigation away, no flow interruption. Verification becomes invisible.

Email Auto-Draft

AI summary generated on email arrival. Reply drafts ready to edit — speed without losing agent control over final output.

See it in action.

A real operational workflow inside Wells Fargo's Omni Channel — from login through call, chat, and email, in one continuous walkthrough.

Conclusion

"That's when design stops being UI and starts being infrastructure."

When design enters a regulated enterprise, the problem changes shape. It's no longer about usability — it's about operational integrity. The Coral Active platform at Wells Fargo demonstrates that structured UX, built from proxy insight and workflow mapping, can reduce compliance exposure while improving agent performance.

The best outcome is a workspace that disappears. When agents stop thinking about the tool and start focusing entirely on the customer, the design has done its job.
